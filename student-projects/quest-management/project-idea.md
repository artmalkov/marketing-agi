# Quest Management Academy — Project Idea

## Project Overview

**Quest Management Academy** — a course for project managers, product managers, and people engaged in ambitious projects. We teach how to lead such projects and deliver results.

## Core Problem

The main problem we see in these clients: **the project is ambitious and incomprehensible**. It's not clear what to do, but it should be done very well.

## Derived Problems

1. **How to work with indeterminacies** — and how to reduce them
2. **How to do something the project sponsor will like** — when there is little time and the result is not guaranteed
3. **How to organize interaction correctly** — between the project manager / team captain on one side, and the project sponsor (investor, budget holder) on the other

## Key Insight: Shared Risk

These two sides must **take the risk together**. If there is no mutual understanding between them:

- The investor will spend money
- The team captain will try very hard but do something wrong
- Everyone will be offended with each other

**Solution:** Build mutual understanding between project managers and sponsors so that both are ready for the fact that the problem may not be solved completely, but only partially.

## Course Structure

We go through all stages:

1. **Primary strategy** of project execution
2. **Uncertainty** — how to work with it
3. **Fighting blockers** that don't allow you to move on
4. **Testing** — which can be delayed because we may discover something we did not foresee at the beginning of the project

All these moments need to be anticipated. In our course, we teach **what leaders and investors should do with them**.
