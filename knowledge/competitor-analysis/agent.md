# Competitor Analysis Agent

## Когда применять агента

Применять когда пользователю нужно найти конкурентов, провести их анализ или собрать генотип конкурентов для сравнения с продуктом.

## Agent Jobs

### Найти конкурентов

**Описание**: Ищет основных конкурентов (10+) для продукта, собирает базовую информацию (название, сайт, выручка, инвестиции и т.д.) и сохраняет в папку competitors студенческого проекта.

**Ситуация**: Нужно определить конкурентное окружение. Источник контекста о продукте — один из: ссылка на студенческий проект (изучить genotype, call-database, landing), явное описание продукта от пользователя, либо список названий/ссылок конкурентов для расширения.

**Примеры запроса**: найди конкурентов, найди конкурентов для проекта, поиск конкурентов, собери список конкурентов, competitor research, find competitors, кто наши конкуренты

**Действия**: выполнить `workflows/find-competitors-rulebook.md`

**Ожидаемый результат**: папка `competitors/` в корне студенческого проекта с `competitors.md` и подпапками по каждому конкуренту.

---

### Глубокий анализ конкурента

**Описание**: Проводит детальный анализ одного или нескольких конкурентов. Материалы складываются в папки конкурентов.

**Ситуация**: Уже есть список конкурентов в competitors/. Нужно углубить понимание по выбранным конкурентам.

**Примеры запроса**: глубокий анализ конкурента, детальный анализ конкурента, проанализируй конкурента, competitor deep dive, deep analysis [competitor name]

**Действия**: выполнить `workflows/deep-analysis-rulebook.md`

**Ожидаемый результат**: обновлённые материалы в папках конкурентов (подпапки в competitors/).

---

### Собрать генотип конкурентов

**Описание**: Формирует competitor-genotype.csv — таблицу потребностей (как в genotype) с колонками по конкурентам. Анализирует сайты и маркетинговые материалы конкурентов, извлекает supply vector (продукты и фичи), матчит с нашим генотипом или дообогащает его новыми потребностями в формате «я хочу…».

**Ситуация**: Есть genotype продукта и материалы по конкурентам. Нужно сравнить, какие потребности закрывают конкуренты.

**Примеры запроса**: собери генотип конкурентов, competitor genotype, competitor-genotype, таблица потребностей конкурентов, сравни генотипы конкурентов

**Действия**: выполнить `workflows/competitor-genotype-rulebook.md`

**Ожидаемый результат**: файл `competitor-genotype.csv` в папке genotype студенческого проекта.
