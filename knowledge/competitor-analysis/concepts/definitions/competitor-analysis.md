# Competitor Analysis

## 1. Общее понимание

**Competitor Analysis** — систематический сбор и анализ информации о конкурентах для понимания их позиционирования, продуктов, сильных и слабых сторон, а также для сравнения с собственным продуктом через призму потребностей клиентов (генотип).

---

## 2. Цели анализа конкурентов

1. **Понимание рынка** — кто основные игроки, как они позиционируются
2. **Сравнение supply vector** — какие потребности закрывают конкуренты своими продуктами и фичами
3. **Выявление пробелов** — потребности, которые мы закрываем, а конкуренты нет (дифференциация)
4. **Обогащение генотипа** — новые потребности, которые конкуренты явно адресуют и мы ещё не учитывали

---

## 3. Competitor Genotype

**Competitor Genotype** — таблица, сопоставляющая потребности (demand vector, формат «я хочу…») с конкурентами. В ячейке стоит 1, если конкурент в своих материалах (сайт, реклама) явно продвигает решение данной потребности; пусто — если нет явного упоминания.

Связь с Product Genotype:
- **Product Genotype** (`knowledge/genotype/concepts/concepts/product-genotype.md`) — наш supply vector: как мы закрываем потребности
- **Competitor Genotype** — supply vector конкурентов: как они закрывают те же потребности

При анализе конкурентов:
- Если фича/сообщение конкурента **совпадает** с потребностью из нашего генотипа → матчинг, ставим 1
- Если фича/сообщение конкурента **новая** (несколько конкурентов её предлагают) → добавляем корневую потребность «я хочу…» в генотип и ставим 1 у соответствующего конкурента

## 4. Структура хранения

```
student-projects/[student-name]/
  competitors/
    competitors.md           ← сводный список конкурентов с базовой информацией
    [competitor-name]/       ← подпапка для детальных материалов по конкуренту
      website.md            ← выдержки с сайта
      marketing.md           ← рекламные материалы, скриншоты описаний
      analysis.md            ← результаты глубокого анализа (по мере появления)
  genotype/
    competitor-genotype.csv  ← таблица потребностей × конкуренты
```

---

## 5. Источники информации о конкурентах

- Официальные сайты
- Рекламные материалы, лендинги
- Рейтинги, обзоры (Crunchbase, SimilarWeb, отраслевые рейтинги)
- Публичные данные: выручка, инвестиции, количество пользователей (если доступно)
